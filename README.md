#### Natural language processing
* nltk

#### Remove punctuation & tokenization
* string
* re

#### Vectorizing
* countvectorizer
* tfidfvectorizer

#### Data visualization
* plotnine

#### Data preprocessing and modeling
* scikit-learn
* numpy
* pandas
* keras
